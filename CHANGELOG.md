## [1.1.0] - 2026-03-08

### Добавлено

#### Модуль обработки изображений с AI супер-разрешением (Real-ESRGAN)
- **Супер-разрешение** на основе Real-ESRGAN для AI-увеличения изображений
- Поддержка 6 моделей:
  - `RealESRGAN_x4plus` — универсальная для фотографий (4x)
  - `RealESRNet_x4plus` — с меньшим уровнем артефактов (4x)
  - `RealESRGAN_x4plus_anime_6B` — оптимизирована для аниме (4x)
  - `RealESRGAN_x2plus` — для 2x увеличения
  - `realesr-animevideov3` — компактная для аниме видео (4x)
  - `realesr-general-x4v3` — универсальная с шумоподавлением (4x)
- Функции: `super_resolution()`, `super_resolution_batch()`, `quick_anime_upscale()`, `quick_photo_upscale()`
- Автоматическая загрузка предобученных моделей
- Поддержка tile-обработки для больших изображений
- Статистика обработки (время, размеры, применённые фильтры)

#### CLI команды для изображений
- `parser image super-resolution` — AI-увеличение с выбором модели
- `parser image models --list-models` — информация о доступных моделях
- Обновлены команды `resize`, `upscale`, `enhance`, `info`, `formats`

#### Python API
- Модуль `parser.image.esrgan` — интеграция Real-ESRGAN
- Обновлён `parser.image.__init__` с экспортом новых функций
- Класс `ImageStats` — статистика обработки изображений

#### Тесты
- Юнит-тесты для модуля изображений (`tests/test_image.py`)
- Интеграционные тесты для Real-ESRGAN (`tests/test_esrgan.py`)
- 23 теста для супер-разрешения

#### Зависимости
- `numpy>=1.24.0` — работа с изображениями
- `opencv-python>=4.5.0` — обработка изображений
- `torch>=2.1.0` — PyTorch для Real-ESRGAN (опционально)
- `torchvision>=0.16.0` — компьютерное зрение (опционально)
- `basicsr>=1.4.2` — базовая библиотека для супер-разрешения (опционально)
- `realesrgan>=0.3.0` — Real-ESRGAN (опционально)

#### Документация
- Обновлён README.md с примерами обработки изображений
- Обновлён QWEN.md с документацией по Real-ESRGAN
- Примеры CLI и Python API

### Изменено

#### Зависимости
- Перемещены зависимости для изображений в `extras_require['image']`
- Обновлена версия до 1.1.0

## [1.0.0] - 2026-03-07

### Добавлено

#### Поддерживаемые форматы
- **Текстовые**: `.txt`, `.md`, `.csv`, `.tsv`, `.log`
- **Данные**: `.json`, `.geojson`, `.yaml`, `.yml`, `.toml`, `.ini`
- **Веб**: `.xml`, `.html`, `.htm`
- **Табличные**: `.xlsx`, `.xlsm`, `.xltx`, `.xltm`, `.ods`
- **Бинарные**: `.bin` (с hex-view)
- **Архивы**: `.zip`, `.tar`, `.gz`, `.tgz`

#### Функции
- Автоматическое определение формата по расширению и содержимому
- Автоопределение кодировки (UTF-8, UTF-16, Windows-1251)
- Парсинг с фильтрацией по ключам, regex, диапазону строк
- Конвертация между форматами (CSV↔XLSX, JSON↔YAML, HTML↔Markdown)
- Пакетная обработка файлов
- Статистика парсинга

#### CLI
- Команда `parse` — парсинг файлов
- Команда `convert` — конвертация форматов
- Команда `analyze` — анализ файлов
- Команда `validate` — валидация файлов
- Команда `formats` — список форматов
- Команда `extract` — распаковка архивов

#### Python API
- Класс `FileParser` — универсальный парсер
- Класс `BaseParser` — базовый класс для парсеров
- Утилиты: `detect_format`, `detect_encoding`, `filter_data`

#### Тесты
- Юнит-тесты для утилит
- Юнит-тесты для CSV, JSON, YAML, XML парсеров
- Интеграционные тесты для FileParser

#### Документация
- README.md с примерами использования
- CONTRIBUTING.md для участников
- CHANGELOG.md
- GitHub Actions workflow для CI/CD
